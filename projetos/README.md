# projetos/

Propósito

Guardar projetos pequenos e completos que juntam vários conceitos (DOM, async, módulos). Exemplos: lista de tarefas, app de notas, fetch-demo.

Como organizar cada projeto

- Cada projeto deve ter sua própria pasta: `projetos/todo-app/`, `projetos/fetch-demo/`.
- Cada projeto pode ter `index.html`, `styles.css`, `app.js` e um `README.md` explicando como executar.

Sugestões

- Criar `todo-app` com persistência em `localStorage` e testes simples.
