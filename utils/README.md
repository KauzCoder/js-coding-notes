# utils/

Propósito

Armazenar funções utilitárias reutilizáveis (por exemplo: `somaArray`, `formatarValor`, `buscarMaior`, etc.) que podem ser importadas por outros exercícios e projetos.

Como usar

- Crie módulos pequenos e bem testados dentro desta pasta e importe-os de outros exemplos.
- Exemplo: `utils/somaArray.js` exporta `somaArray` e `utils/formatar.js` exporta formatadores.
