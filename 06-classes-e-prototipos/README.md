# classes-e-prototipos/

Propósito

Esta pasta contém exemplos de classes ES6 e do sistema de protótipos do JavaScript: declaração de classes, construtores, herança, métodos estáticos, e como o sistema de protótipos funciona por baixo dos panos.

Como usar

- Crie arquivos como `Pessoa.js`, `Aluno.js` mostrando herança e polimorfismo.
- Compare implementações usando `class` e configuração manual do `prototype`.

Exercícios

- Criar classe `ContaBancaria` com depósitos, saques e histórico.
- Implementar herança (ex.: `ContaPoupanca` estende `ContaBancaria`).
