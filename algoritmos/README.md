# algoritmos/

Propósito

Exercícios e implementações clássicas de algoritmos e estruturas de dados: ordenação (bubble, quicksort), busca (linear, binária), pilha, fila e estruturas simples.

Como usar

- Cada algoritmo em um arquivo separado (`bubbleSort.js`, `binarySearch.js`) com comentários e casos de teste simples.
- Criar um `README` por exercício explicando a complexidade e uso.
