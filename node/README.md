# node/

Propósito

Exemplos e exercícios para Node.js: scripts simples, uso do `fs`, criação de um servidor HTTP básico, leitura/escrita de arquivos, e exemplos de pacotes npm.

Como usar

- Inicialize um `package.json` com `npm init -y` quando criar um projeto na pasta.
- Exemplos: `server.js` (http), `leArquivo.js` (fs.readFile), `escreveArquivo.js`.

Sugestões

- Criar um script que lê um JSON de tarefas e atualiza (ex.: marca tarefa como completa).
- Exemplo de CLI simples usando `process.argv`.
