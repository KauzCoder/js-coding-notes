# tests/

Propósito

Guardar testes automatizados (unidades) para suas funções e módulos. Use frameworks como Jest ou Mocha+Chai.

Como usar

- Instale Jest localmente (`npm install --save-dev jest`) em um projeto que use `package.json`.
- Adicione scripts em `package.json`: `"test": "jest"`.
- Coloque testes em arquivos com sufixo `.test.js` ou `.spec.js`.

Sugestões

- Criar testes para funções utilitárias como `somaArray`, `calculaMediaGasto`.
- Exemplos simples: testar valores de entrada válidos e casos de borda.
