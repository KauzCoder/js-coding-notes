# dom/

Propósito

Esta pasta deve conter exemplos e exercícios relacionados à manipulação do DOM e eventos no navegador: seleção de elementos, alteração de atributos/estilos, criação e remoção de elementos, tratamento de formulários e eventos (click, submit, input), além de pequenos projetos front-end (to-do list, contador, sliders).

Como usar

- Crie arquivos HTML + JS dentro desta pasta (por exemplo: `exemplo-dom.html` e `exemplo-dom.js`).
- Abra o arquivo HTML no navegador ou rode um servidor local (por exemplo, `npx http-server` ou `Live Server` no VS Code) para testar interações.
- Mantenha exemplos pequenos e documentados para facilitar o estudo.

Sugestões de exercícios

- Manipular uma lista (adicionar/remover itens com botões).
- Formulário que valida campos e mostra mensagens.
- Projeto pequeno: TODO com persistência em localStorage.
